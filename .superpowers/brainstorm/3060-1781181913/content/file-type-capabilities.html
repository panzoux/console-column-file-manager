🔍 File type detection — the "rifle system"

A fast lookup combining magic bytes (first 16 bytes) + file extension fallback. Covers 80+ formats: images, video, audio, archives, databases, fonts, documents, executables, scripts. Produces a human-readable label like JPEG Image, SQLite 3 Database, Gzip Archive. Pure C#, zero allocations for the common path.
