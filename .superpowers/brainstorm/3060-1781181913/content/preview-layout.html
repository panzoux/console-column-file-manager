Preview Pane Layout

When a file is selected, where does the preview appear and how wide is it?

┌──────────────┬──────────────┬──────────────┬──────────────┐
│ Drives       │ C:\          │ Users        │ ▓ Preview ▓▓▓│
├──────────────┼──────────────┼──────────────┼──────────────┤
│ C:\          │ Program Files│ Desktop/     │ readme.txt   │

│              │ pagefile.sys │ video.mp4    │ Modified:    │

A — Fixed width (same as columns)

Preview pane is exactly one column wide (32 chars). Clean and uniform. Limited space for content — text wraps aggressively. Works in narrow terminals.

┌──────────────┬──────────────┬──────────────┬──────────────────────────────┐
│ Drives       │ C:\          │ Users        │ ▓▓▓▓▓▓ Preview ▓▓▓▓▓▓▓▓▓▓▓▓▓│
├──────────────┼──────────────┼──────────────┼──────────────────────────────┤
│ C:\          │ Program Files│ Desktop/     │ readme.txt  txt • 2.4 KB    │
│ D:\          │ Users/       │ > readme.txt │ Modified: 2026-01-15 09:32   │
│ E:\          │ Windows/     │ photo.jpg    │ ──────────────────────────── │
│              │ pagefile.sys │ video.mp4    │ Hello World!                 │
│              │              │ archive.zip  │ This is a sample readme.     │
│              │              │              │ It has several lines of text  │
│              │              │              │ that demonstrate the preview  │
│              │              │              │ wrapping behavior nicely here.│
│              │              │              │ Line 9 of the file content.  │

B — Full remaining width (recommended)

Preview pane expands to fill ALL remaining terminal width. More room for text content, hex dumps, image info. Hides automatically if remaining width < 30 chars.

┌──────────────┬──────────────┬───────────────────────────────────────────┐
│ Drives       │ C:\          │ ▓▓▓▓▓▓▓▓▓▓▓▓▓▓ Preview ▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓│
├──────────────┼──────────────┼───────────────────────────────────────────┤
│ C:\          │ Program Files│ readme.txt  Plain Text • 2.4 KB          │
│ D:\          │ > Users/     │ C:\Users\readme.txt                       │
│ E:\          │ Windows/     │ Modified: 2026-01-15 09:32  Read-only: No │
│              │ pagefile.sys │ ─────────────────────────────────────────│
│              │              │ Hello World!                              │
│              │              │ This is a sample readme file.             │
│              │              │ Demonstrating wider preview pane layout.  │
│              │              │ More text fits without wrapping.          │
│              │              │ Line 5 of the file content shown here.   │
└──────────────┴──────────────┴───────────────────────────────────────────┘

C — Split: preview replaces rightmost dir column

When a directory is selected, preview appears to the right at full remaining width. Fewer visible directory columns, but maximum preview space. Better for wide terminals.
